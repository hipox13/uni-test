# unicef-redev-cms

**Admin CMS** for UNICEF Donation Platform. React 18 + Vite + TanStack Query.  
SPA for dashboard, donations, transactions, reports, users, settings. Caching via TanStack Query.

## Structure

- `src/features/` – dashboard, donations, transactions, reports, users, settings
- `src/components/` – layout, common, ui
- `src/lib/` – api client, hooks, utils
- `src/store/` – Zustand
- `src/types/` – shared types

## Scripts

- `npm run dev` – dev server (default port 5174)
- `npm run build` – production build

## Repo role

Public website is **unicef-redev-web**. This repo is admin only.
